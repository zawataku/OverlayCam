# OverlayCam - オーバーレイカメラ

OverlayCamは、カメラ映像とギャラリーの画像を重ねて写真を撮影することができるアプリです。

## 主な機能

*   **オーバーレイ画像の選択:** ギャラリーから画像を選択
*   **オーバーレイ画像の変形:** 
    *   2本指でのプレビュー画像拡大・縮小（ピンチズーム）
    *   2本指でのプレビュー画像回転
*   **撮影:** 
    *   カメラのプレビューにオーバーレイ画像を重ねて写真を撮影
    *   撮影した写真をギャラリーに保存

## 開発環境

*   Flutter 3.16.1
*   Dart 2.19.2
